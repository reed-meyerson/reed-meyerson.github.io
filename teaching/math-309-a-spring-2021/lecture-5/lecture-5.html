General \(n\times n\) IVP

\(x(t)=(x_1(t),x_2(t),...,x_n(t))\). Let \(A\) be an \(n\times n\) matrix.

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=x_0 \end{cases} \end{equation*}
  • Initial value problem (IVP)

Observation

If \(A\) is diagonal, this is easy to solve.

Example

Let \(x(t)=(x_1(t),x_2(t))\)

\begin{equation*} \begin{cases} x' &=\begin{pmatrix} a & 0\\ 0 & d\end{pmatrix} x\\ x(0)&=(C_1,C_2) \end{cases} \end{equation*}

Two Regular ODEs

\begin{equation*} \begin{cases} x_1'&=ax_1\\ x_1(0)&=C_1 \end{cases} \end{equation*}

\(\implies x_1(t)=C_1e^{at}\)

\begin{equation*} \begin{cases} x_2'&=dx_2\\ x_2(0)&=C_2 \end{cases} \end{equation*}

\(\implies x_2(t)=C_2e^{dt}\)

  • \(x(t)=(C_1e^{at},C_2e^{bt})\)

Common Mathematical Idea

Things that are trivial for diagonal matrices are often possible for diagonalizable matrices.

Solving a Diagonalizable \(2\times 2\) ODE

Let \(x(t)=(x_1(t),x_2(t))\) and \(\{v_1,v_2\}\) be a basis for \(\mathbb{R}^2\). Consider

\begin{equation*} \begin{cases} x' &=(v_1|v_2)\begin{pmatrix} a & 0\\ 0 & d\end{pmatrix}(v_1|v_2)^{-1}x\\ x(0)&=(C_1,C_2) \end{cases} \end{equation*}

Multiply on left by \((v_1|v_2)^{-1}\).

New ODE

\begin{equation*} \begin{cases} (v_1|v_2)^{-1}x' &=\begin{pmatrix} a & 0\\ 0 & d\end{pmatrix}(v_1|v_2)^{-1}x\\ (v_1|v_2)^{-1}x(0)&=(v_1|v_2)^{-1}(C_1,C_2) \end{cases} \end{equation*}
  • Define \(\tilde x(t)=(v_1|v_2)^{-1}x(t)\)
  • Define \((\tilde C_1,\tilde C_2)=(v_1|v_2)^{-1}(C_1,C_2)\).

Then \(x\) satisfies original ODE if and only if \(\tilde x\) satisfies

\begin{equation*} \begin{cases} \tilde x' &=\begin{pmatrix} a & 0\\ 0 & d\end{pmatrix}\tilde x\\ \tilde x(0)&=(\tilde C_1,\tilde C_2) \end{cases} \end{equation*}

We Assumed

  • \((Bx)'(t)=B[x'(t)]\) when \(B\) is a constant matrix
  • Exercise: Convince yourself that this is true.

Solve diagonal ODE

  • \(\tilde x(t)=(\tilde C_1e^{at},\tilde C_2e^{dt})\)

Now that we know \(\tilde x\), transform back to get \(x\).

\begin{align*} (v_1|v_2)^{-1}x(t)&=\tilde x(t)\\ x(t)&=(v_1|v_2)\tilde x(t)\\ &=(v_1|v_2)(\tilde C_1e^{at},\tilde C_2e^{dt}) \end{align*}

In General

Let \(A=PDP^{-1}\) be an \(n\times n\) diagonalizable matrix. To solve

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=x_0 \end{cases} \end{equation*}

Rewrite as

\begin{equation*} \begin{cases} x'&=PDP^{-1}x\\ x(0)&=x_0 \end{cases} \end{equation*}

Change Coordinates

  • Define \(\tilde x(t)=P^{-1}x(t)\)
  • Define \(\tilde x_0=P^{-1} x_0\).
\begin{equation*} \begin{cases} \tilde x'&=D\tilde x\\ \tilde x(0)&=\tilde x_0 \end{cases} \end{equation*}

Solve the diagonal system

Change Back

If \(\tilde x(t)\) solve the system on the previous slide, then \(x(t)=P\tilde x(t)\) solves the original system.

Different Perspective

Suppose \(A v_1=\lambda_1 v_1\), and

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=v_1 \end{cases} \end{equation*}

Claim: \(x(t)=e^{\lambda_1t}v_1\)

Check Initial Condition

\(x(0)=e^0v_1=v_1\)

Check ODE

\begin{align*} x'(t)&=\frac{d}{dt}[e^{\lambda_1t}v_1]\\ &=\lambda_1 e^{\lambda_1t}v_1 \end{align*}

And

\begin{align*} Ax(t)&=A[e^{\lambda_1t}v_1]\\ &=e^{\lambda_1t}[Av_1]\\ &=\lambda_1e^{\lambda_1t}v_1 \end{align*}

So \(x'=Ax(t)\)

We Assumed

  • If \(B\) is a constant matrix, \(y\) is a constant vector and \(f\in C^1(\mathbb{R})\), then

    \begin{equation*} B[f(t)y]=f(t)[By] \end{equation*}

    And

    \begin{equation*} \frac{d}{dt}[f(t)y]=f'(t)y \end{equation*}
  • Exercise: Convince yourself that this is true.

What does this do for us?

If \(\{v_1,...,v_n\}\) is an eigenbasis for \(A\), then we can solve

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=v_k \end{cases} \end{equation*}

Only get \(n\) solutions? So what?

Linear Combinations of Solutions are Solutions

Claim: The set \(\{x\in C^1(\mathbb{R};\mathbb{R}^n)|x'=Ax\}\) is a vector space.

Proof

Let \(x,y\) be such that \(x'=Ax\) and \(y'=Ay\). Let \(\alpha\in\mathbb{R}\). Then

\begin{align*} A(\alpha x+y)(t)&=\alpha Ax(t)+Ay(t)\\ &=\alpha x'(t)+y'(t)\\ &=(\alpha x+y)'(t) \end{align*}

Linear Combinations with Initial Conditions

Let \(y_k(t)\) solve

\begin{equation*} \begin{cases} y_k'&=Ay_k\\ y_k(0)&=v_k \end{cases} \end{equation*}

Then the solution to

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=c_1v_1+c_2v_2+...+c_nv_n \end{cases} \end{equation*}

Is \(x(t)=c_1y_1(t)+c_2y_2(t)+...+c_ny_n(t)\)

Proof

We already know \(x'=Ax\). Just need to check initial condition.

\begin{align*} x(0)&=c_1y_1(0)+c_2y_2(0)+...+c_ny_n(0)\\ &=c_1v_1+c_2v_2+...+c_nv_n \end{align*}

Example

Let \(A=\begin{pmatrix} 8 & -3/2\\ 4 & 3\end{pmatrix}\). Solve

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=(7,8) \end{cases} \end{equation*}

(1.) Diagonalize

\(A=\begin{pmatrix} 1 & 3 \\ 2 & 4\end{pmatrix}\begin{pmatrix}5 & 0\\ 0 & 6\end{pmatrix}\begin{pmatrix} 1 & 3 \\ 2 & 4\end{pmatrix}^{-1}\)

(2.) Eigenvector ODEs

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=(1,2) \end{cases} \end{equation*}

\(\implies x(t)=e^{5t}(1,2)\)

\begin{equation*} \begin{cases} x'&=Ax\\ x(0)&=(3,4) \end{cases} \end{equation*}

\(\implies x(t)=e^{6t}(3,4)\)

(3.) Solve for Initial Condition

Solve \(c_1(1,2)+c_2(3,4)=(7,8)\)

  • \(-2(1,2)+3(3,4)=(7,8)\)

(4.) Write Down Solution

\(x(t)=-2e^{5t}(1,2)+3e^{6t}(3,4)\)
