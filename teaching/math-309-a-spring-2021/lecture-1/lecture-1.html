Ordinary Differential Equations (ODEs)

  • System evolving in time
  • State of system can be described by a finite number of values

Example: orbits of planets

  • state = xyz coordinates of planets

History

  • Newton and Liebniz (1670s)
  • \(\vec{F}=m\vec{a}\implies \vec{F}(t)=m\vec{x}''(t)\)
  • Lots more people

ODE Example

\begin{equation*} \begin{cases} x_1'(t)&=x_1(t)+2x_2(t)\\ x_2'(t)&=3x_1(t)+4x_2(t)\\ (x_1(0),x_2(0))&=(5,6) \end{cases} \end{equation*}

Partial Differential Equations (PDEs)

  • System evolving in time
  • State of system can be described by function

Example: temperature of wire changing in time

  • State = temperature of wire

Example: vibrating string

  • State = position & velocity of string

History

  • Wave Equation (d’Alembert, Euler, Bernouli 1750s/1760s)
    • Vibrating string/vibrating membrane/acoustic vibration
  • Laplace’s Equation (Laplace 1780s)
    • Gravitational potential field
  • Heat Equation (Fourier 1810s/1820s)
    • Heat flow in wire/solid/atmosphere

PDE Example

\begin{equation*} \begin{cases} \partial_tu(x,t)-\partial_x^2u(x,t)&=0 &\text{ }x\in[0,1],t>0\\ u(x,0)&=x^2 &\text{ }x\in[0,1]\\ u(0,t)=u(1,t)&=0 &\text{ }t>0 \end{cases} \end{equation*}

    \begin{align*}x(t)&=(x_1(t),y_2(t))\\y(t)&=(y_1(t),y_2(t))\end{align*}

    then

    \begin{equation*} (x+y)(t)=(x_1(t)+y_1(t),x_2(t)+y_2(t)) \end{equation*}

We Can Add…

  • matrices

    \begin{equation*} \begin{pmatrix} a & b\\ c & d\end{pmatrix}+\begin{pmatrix} e & f\\ g & h\end{pmatrix} = \begin{pmatrix} a+e & b+f\\ c+g & d+h\end{pmatrix} \end{equation*}

Solving an actual ODE

From 307

\begin{equation*} \begin{cases} x_1'&=1x_1\\ x_1(0)&=3\\ \end{cases} \end{equation*}

\(x_1(t)=3e^{t}\)

Solving an actual ODE

\begin{equation*} \begin{cases} x_2'&=2x_2\\ x_2(0)&=4 \end{cases} \end{equation*}

\(x_2(t)=4e^{2t}\)

Solving a harder ODE

\begin{equation*} \begin{cases} x_1'&=&3x_1+&2x_2\\ x_2'&=& &1x_2\\ x_1(0)&=&4\\ x_2(0)&=&5 \end{cases} \end{equation*}

Solving a harder ODE

\begin{equation*} \begin{cases} x_1'&=&3x_1+&2x_2\\ x_2'&=& &1x_2\\ x_1(0)&=&4\\ x_2(0)&=&5 \end{cases} \end{equation*}
\begin{equation*} \implies \begin{cases} x_2'&=1x_2\\ x_2(0)&=5 \end{cases} \end{equation*}

So, by 307 we have \(x_2(t)=5e^t\)

Solving a harder ODE

\begin{equation*} x_2(t)=5e^t\text{ and } \begin{cases} x_1'&=&3x_1+&2x_2\\ x_2'&=& &1x_2\\ x_1(0)&=&4\\ x_2(0)&=&5 \end{cases} \end{equation*}
\begin{equation*} \implies \begin{cases} x_1'&=&3x_1+&10e^t\\ x_1(0)&=&4\\ \end{cases} \end{equation*}

By 307 we get \(x_1(t)=9e^{3t}-5e^t\)

Substitution Method
